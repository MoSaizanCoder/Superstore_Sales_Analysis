## 📌 Project Overview
This project is an interactive Explanatory Data Analysis (EDA) dashboard built to identify loss leakage patterns and optimize revenue for a retail superstore. Analyzing over 10,000 sales records, this application categorizes insights into Univariate, Bivariate, and Multivariate analysis, providing clear business context for every visualization.

## 🚀 Key Features
* **Interactive Filtering:** Dynamic sidebar allowing users to filter sales data by specific years.
* **Key Performance Indicators (KPIs):** Top-level metrics displaying Total Sales, Net Profit, and Total Loss Leakage at a glance.
* **Structured EDA:**
  * **Univariate Analysis:** Understand individual variable distributions (Sales, Profit, Orders by Category/Region).
  * **Bivariate Analysis:** Explore relationships to find profitability drivers and leakage points (Sales vs. Profit, Discount Impacts).
  * **Multivariate Analysis:** Deep dive into complex interactions using correlation matrices and time-series trends.
* **Actionable Business Insights:** Every chart includes an expandable "Explanation" section detailing the strategic takeaway.
